Importing Libraries

The first step to working in an IPython Notebook is importing the necessary libraries you need. Typically, I always import pandas and numpy, and add others as needed.

#General syntax to import a library: 
##import (library) as (give the library a nickname/alias)
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
